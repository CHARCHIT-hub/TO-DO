# ✅ TO-DO App

A modern To-Do list app built with React, TypeScript and Tailwind CSS.

## 🌐 Live Demo
[Click here to view the app](https://to-do-k-s-projects21.vercel.app/)

## 🛠️ Built With
- React
- TypeScript
- Tailwind CSS
- Vite

## 🚀 Run Locally

1. Clone the project
   git clone https://github.com/CHARCHIT-hub/TO-DO.git

2. Go to project folder
   cd TO-DO

3. Install dependencies
   npm install

4. Start the app
   npm run dev

5. Open http://localhost:5173 in your browser
